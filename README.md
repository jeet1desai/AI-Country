# Country-Weather 

### Requirement
copy paste

### create project
npx create-react-app . --template typescript

### installed library

1) react-router-dom : v^5.2.0 : for navigation

2) axios: v^0.24.0 : for data fetch / HTTP request

3) @material-ui/core : v^5.2.3 : for styling 


### To run
npm start

### To test 
npm run test
